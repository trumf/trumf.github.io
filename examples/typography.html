Color Variations

Text colors are managed through CSS variables to maintain consistency and support theming.

--text-primary

Primary Text

This is the main text color used for headings and body text. It has the highest contrast against the background.

--text-secondary

Secondary Text

Secondary text is used for less emphasized content while still maintaining good readability.

--text-tertiary

Tertiary Text

Tertiary text is used for supplementary information, metadata, and less important content.

Responsive Behavior

Resize your browser window to see how typography adapts to different screen sizes.

Responsive heading

This heading will change size

The typography system includes media queries that adjust text sizes at these breakpoints:

  • Desktop (default): Full-size scale
  • Large Tablets (max-width: 1200px): Slight reduction for largest sizes
  • Tablets (max-width: 768px): Medium reduction
  • Mobile (max-width: 480px): Significant reduction
